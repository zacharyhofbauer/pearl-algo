PEARLALGO FUTURES TRADING - QUICK REFERENCE
===========================================

DASHBOARD COMMANDS
------------------
python scripts/status_dashboard.py          # Show dashboard once
python scripts/status_dashboard.py --live   # Live updating dashboard (5s refresh)

GATEWAY MANAGEMENT
------------------
pearlalgo gateway status                     # Check gateway status
pearlalgo gateway start                      # Start IB Gateway
pearlalgo gateway stop                       # Stop IB Gateway
pearlalgo gateway restart                     # Restart IB Gateway

TRADING COMMANDS
----------------
pearlalgo trade auto ES NQ GC --strategy sr --interval 300
pearlalgo trade auto MGC MYM MCL MNQ MES --strategy sr --interval 60 --tiny-size 3

SYSTEM STATUS
-------------
pearlalgo status                             # Overall system status
python scripts/system_health_check.py       # Detailed health check
python scripts/test_broker_connection.py     # Test IB connection

DASHBOARD METRICS QUICK REFERENCE
---------------------------------

Risk States:
  ✅ OK          - Drawdown < 50% of daily loss limit
  ⚠️  NEAR_LIMIT - Drawdown 50-80% of daily loss limit
  ❌ HARD_STOP   - Drawdown > 80% or trading halted

Key Metrics:
  Sharpe Ratio   - Risk-adjusted returns (higher = better)
  Sortino Ratio  - Downside risk-adjusted returns (higher = better)
  Drawdown %     - Percentage of daily loss limit used
  Realized P&L   - Profit/loss from closed positions
  Unrealized P&L - Current profit/loss from open positions

Per-Symbol Columns:
  Symbol         - Trading symbol (ES, NQ, GC, etc.)
  Last Signal    - Time of most recent signal
  Side           - LONG/SHORT/FLAT direction
  Realized P&L   - Cumulative realized profit/loss
  Unrealized P&L - Current unrealized profit/loss
  Position       - Current position size
  Trades         - Number of trades today
  Max            - Maximum contracts allowed

Trade Statistics:
  Total Trades   - Number of completed trades
  Win Rate       - Percentage of winning trades
  Avg Hold Time  - Average time in trade (minutes)
  Largest Winner - Best trade P&L
  Largest Loser  - Worst trade P&L

FILE LOCATIONS
--------------
Signals:        signals/YYYYMMDD_signals.csv
Reports:        reports/YYYYMMDD_report.md
Performance:    data/performance/futures_decisions.csv
Config:         config/prop_profile.yaml

LOGS
----
tail -f logs/micro_trading.log               # Micro strategy trading log
tail -f logs/micro_console.log               # Micro strategy console output
tail -f logs/standard_trading.log            # Standard strategy trading log

STOP TRADING
------------
bash scripts/kill_my_processes.sh            # Kill all trading processes
pkill -f "pearlalgo trade auto"              # Manual kill

TROUBLESHOOTING
---------------
Gateway not starting?
  - Check: pearlalgo gateway status
  - Restart: pearlalgo gateway restart
  - Check logs: tail -50 /tmp/ibgateway.log

Can't connect?
  - Test: python scripts/test_broker_connection.py
  - Health: python scripts/system_health_check.py

Dashboard shows no data?
  - Check: data/performance/futures_decisions.csv exists
  - Verify: Trading has been executed today
  - Check: File permissions

Risk calculations wrong?
  - Verify: config/prop_profile.yaml settings
  - Check: Daily loss limit matches expectations
  - Review: Performance log data quality
