# Docs Archive

This folder holds documentation that is useful for historical context, rollout
forensics, or old implementation notes, but should not be treated as the current
operational source of truth.

## What belongs here

- completed deployment notes
- one-off implementation writeups
- superseded environment/setup notes
- dated journal entries
- older architecture snapshots that no longer match the tracked tree

## What does not belong here

- current operator instructions
- current path/runtime truth
- current lifecycle commands
- current config guidance

## Current truth lives in

- `docs/START_HERE.md`
- `docs/CURRENT_OPERATING_MODEL.md`
- `docs/PATH_TRUTH_TABLE.md`
- `docs/GATEWAY.md`
- `docs/TESTING_GUIDE.md`
