# Telegram AI Menu - Simplified

## What Changed

The Telegram AI menu has been **simplified** to focus on what you actually need: **getting suggestions on-demand with clear explanations and easy application**.

## New `/suggest` Command

### Usage
```
/suggest          - Show active suggestions
/suggest new      - Generate fresh suggestions (runs analysis)
/suggest refresh  - Same as 'new'
```

### What You Get

Each suggestion shows:

1. **WHAT** - What config/setting is being changed
   - Current value → Suggested value
   - Clear config path

2. **WHY** - Why this change is recommended
   - Rationale based on current performance
   - Data-driven reasoning

3. **WHEN** - When the suggestion was created
   - Timestamp for context

4. **Risk Level** - 🟢 Low / 🟡 Medium / 🔴 High

### Easy Actions

- **✅ Apply** button - Apply suggestion with one tap
- **🔍 Preview** button - See full details before applying
- **🔄 Generate New** - Run fresh analysis anytime
- **📋 All Suggestions** - See complete list (if more than 5)

## Simplified AI Monitor Menu

**Before (Confusing):**
- 📊 Analyze Now
- 📈 Signals
- 🔧 System  
- 📉 Market
- 💡 Suggest
- 📋 Suggestions

**After (Simple):**
- 💡 **Get Suggestions** - Main action you need
- 📊 **Quick Analysis** - Optional deep dive
- ▶️/⏹️ **Start/Stop Monitor** - Control monitor
- 🔄 **Refresh** - Refresh status

## How to Use

### Step 1: Get Suggestions
```
/suggest
```
or tap: **AI Hub** → **🔍 AI Monitor** → **💡 Get Suggestions**

### Step 2: Review
Each suggestion shows:
- **WHAT** config is changing
- **WHY** it's recommended (based on your performance data)
- **WHEN** it was suggested

### Step 3: Apply
Tap **✅ Apply** button next to the suggestion you want to apply.

### Step 4: Rollback (if needed)
If something goes wrong, use the **🔄 Rollback** button that appears after applying.

## Rollback

After applying, you'll see:
```
✅ Suggestion Applied

Widen Stop Losses

Config: risk.stop_loss_atr_multiplier
Changed: 1.5 → 2.5

Request ID: req_abc123

🔄 Rollback: /rollback_suggestion req_abc123
```

Tap **🔄 Rollback** to undo if needed.

## Old Commands (Still Work)

- `/suggestions` - Full list (more detailed, paginated)
- `/analyze_now` - Force full analysis
- `/apply_suggestion <id>` - Apply via command
- `/rollback_suggestion <id>` - Rollback via command

But **you probably won't need them** - the new `/suggest` command covers everything.

## Benefits

✅ **Simple** - One command does everything  
✅ **Clear** - WHY/WHEN/WHAT format  
✅ **Fast** - Tap to apply, no typing  
✅ **Safe** - Preview before applying  
✅ **Flexible** - Generate new suggestions anytime  

## Configuration

The simplified menu is automatically enabled. No config changes needed.

The suggestions are generated by Claude Monitor using your actual performance data, so they're relevant and actionable.
